Test class used to mimic an API exposing some application logic via SOAP using RPC/Literal WSDL.

NOTE: The AgilePHP #@WebMethod annotation authorizes a method for inclusion during WSDL generation.

The AgilePHP #@WSDL interceptor uses the data types specified in the PHP-doc comment blocks (specifically the @param and @return annotations) during WSDL generation. If these types are not present, the #@WSDL generator will use xsd:anyType.
