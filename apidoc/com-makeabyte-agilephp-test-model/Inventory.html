Class Inventory

Description
Description | Methods (details)

The inventory domain model object responsible for maintaining ActiveRecord state for a specified inventory item. Note the use of the #@Id interceptor which performs a SQL lookup in the database and populates the ActiveRecord state when the mutator method is called.
