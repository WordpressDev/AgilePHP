		createIV (line 244)
	
 
	
	

Creates an IV suitable for the specified cipher using the MCRYPT_MODE_CBC module.


NOTE: You can use this method to create an IV for use in agilephp.xml for the Crypto  	   component to use. Simply base64_encode the return value and place it in the crypto  	   components iv attribute in agilephp.xml.

	
    
				
  • return: The initial value (iv) as created by mcrypt_create_iv.
