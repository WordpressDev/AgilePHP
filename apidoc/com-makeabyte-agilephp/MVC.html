		processRequest (line 193)
	
 
	
	

Parses the current request URI to obtain the controller, action method, and arguments  present for this request and then performs the invocation. If these parameters ARE NOT  present, the default controller and default action method are used instead.


NOTE: The URI requirement to communicate with this MVC system is as follows        http://domain.com/ScriptName.php/ControllerName/ActionMethod/arg1/arg2/arg3/etc...
