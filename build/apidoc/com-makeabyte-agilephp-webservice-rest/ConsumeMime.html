Class ConsumeMime

Description
Description | Vars (details)

Tells the REST request processing interceptor (#@GET|#@POST|#@PUT|#@DELETE) to transform received HTTP data into a PHP data type before passing into the REST service method. Conversions occur based on the specified #@ConsumeMime::type as follows:

1) application/xml = SimpleXMLElement 2) application/json = json_decoded data 3) application/x-yaml = yaml_parsed data 4) application/xhtml+xml = raw data passed from the client
