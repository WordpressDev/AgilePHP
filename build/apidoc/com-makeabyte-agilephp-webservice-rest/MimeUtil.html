		static negotiate (line 49)
	
 
	
	

Responsible for negotiating a mime type used to send data from the server


to the client. This is done by first checking the client preferred mime types  and ensuring one of the following mimes is supported:  1) application/xml  2) application/json  3) application/xhtml+xml
The first mime type to match according to the clients preferred list is used.  Finally, the REST service action method is introspected in search of a #@ProduceMime  annotation. If #@ProduceMime::type is accepted by the client this mime type is returned,  otherwise a 406 "Not Acceptable" is returned to the client.

	
    
				
  • return: The negotiated mime type
