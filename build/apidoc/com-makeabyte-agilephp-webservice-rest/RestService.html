Class RestService

Description
Description | Methods (details)

Interceptor responsible for capturing REST resource requests from the MVC component and routing them to the proper service method and resource based on URI composition.

NOTE: It seems that ironically enough, REST does not really fit the MVC paradigm, since MVC requests use /service-or-controller/action-also-known-as-a-verb whereas REST requests use /service-or-controller/nouns-only. The #@RestService acts as a router or front controller to ensure requests to REST resources stay "restful".
